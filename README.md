# UART
> This project implements a full-duplex UART controller with independent transmit and receive paths, FIFO buffering, baud-rate generation, and a register-based control interface

## RTL Modules

```text
rtl
├── bus
├── common
├── top
└── uart
```

## Objectives
> Demonstrate the ability to:
>
> - Finite State Machine (FSM) design.
> - Counter-based timing generation.
> - FIFO-based data buffering.
> - Clocked synchronous logic.
> - Status and control register immplementation.
> - RTL simulation and verification.
> - Basic hardware/software interaction.
